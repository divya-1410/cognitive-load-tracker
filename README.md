# 🧠 Cognitive Load Tracker

## 📌 Overview
The Cognitive Load Tracker is a smart study analysis system that monitors a student's focus levels over time and identifies cognitive fatigue patterns. Unlike traditional timers, this system focuses on mental efficiency rather than just study duration.

## 🎯 Objective
To help students optimize their study sessions by detecting when their learning efficiency decreases and suggesting better break timings.

## 🚀 Features
- 📊 Focus level tracking (1–5 scale)
- ⏱ Time-based session monitoring
- ⚠️ Fatigue detection
- 🔥 Peak performance identification
- 📈 Graph visualization
- 💻 GUI (Tkinter)
- 🌐 Web app (Streamlit)

## 🛠 Tech Stack
- Python
- Tkinter (GUI)
- Matplotlib (Graphs)
- Streamlit (Web Deployment)

## 📷 Output
- Graph showing focus vs time
- Session analysis with:
  - Average focus
  - Fatigue point
  - Suggestions
